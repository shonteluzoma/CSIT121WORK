Welcome to my webpage! At first this task seemed like a lot
but once you start to get in the groove of things it's really not that bad.
Having the basic understanding of how to structure HTML is important
in teaching you how webpages are built. Once you know your tags and
structures and work on your precision while typing to avoid errors, it should
be smooth sailing from there.

What I Learned...

  1. Attention to detail is key, always check for errors and typos.
  2. An ancestor refers to any element that is connected but further up the document tree - no matter how many levels higher.
  3. A descendant refers to any element that is connected but lower down the document tree - no matter how many levels lower.

Lab 3

Learning about CSS was fun. It really enhances a webpage visually and bring life to the html. I also love how we can link it to our html externally because it makes everything way more organized that way. Overall it's a good way to express your creativity through your website.

Things that I learned since Lab 02 for Lab 03

  1. CSS lets you pick which HTML elements to style using selectors.
  2. CSS stands for Cascading Style Sheets
  3. CSS saves a lot of work. It can control the layout of multiple web pages all at once
